## Installation Steps

Open CMD where you want to clone the project & run the following commands

```
git clone https://github.com/Ahmed-nd/spyware-detector.git
cd spyware-detector
python server.py
```

## Python Packages

* flask v2.0.3
* werkzeug v2.0.3
* joblib 
* numpy
* pefile
